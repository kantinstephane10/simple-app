What this is

A small dashboard for keeping one or more counters, built to exercise a full build-and-deploy pipeline: Spring Boot on the backend, vanilla JavaScript on the front end, packaged by Maven and shipped through CodeBuild, CodeDeploy/EC2, and Elastic Beanstalk.

Keyboard shortcuts

These apply to whichever counter you last interacted with:

  • ↑ / → — increment
  • ↓ / ← — decrement
  • R — reset

Double-click a counter's name to rename it.

How state works

Counters live in your browser session, not a database — they reset if your session expires or you switch browsers. Each counter tracks its own current value, best value reached, and total click count, and every action is recorded in the activity feed on the main page.

Tech stack

  • Java 17 + Spring Boot (Web, Thymeleaf, Actuator)
  • Vanilla JavaScript — no frontend framework or build step
  • Maven for packaging
  • OpenAPI/Swagger docs for the REST API
